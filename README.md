# webpack-react-wechat
> Wechat h5 page based on webpack and react.

## get-start:
```bash
git clone https://github.com/afeiship/webpack-react-wechat.git
npm install 
npm run build
npm run start
```

## feature:
+ [x] spa 
+ [x] webpack 3.x
+ [x] next-redux-base
+ [x] image compress
+ [x] fonts/images to folder
